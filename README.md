# 🌊 LocalFlow

**Build local, private AI agents visually. No cloud. No API keys. No data leaks.**

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg)](http://makeapullrequest.com)

LocalFlow is an open-source, node-based visual builder for local LLMs. It combines the drag-and-drop ease of Zapier with the privacy of local AI models (via Ollama). 

## ✨ Features
* 🔒 **100% Private:** Runs entirely on your local machine.
* 🧩 **Visual Builder:** Drag, drop, and connect nodes.
* 🦙 **Ollama Integration:** Native support for Llama 3, Mistral, and more.

## 🚀 Quick Start
1. Install [Ollama](https://ollama.com/)
2. Clone and run:
`npm install && npm run dev`
